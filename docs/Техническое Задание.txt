Веб-приложение по сбалансированному питанию

1. Личный кабинет пользователя содержит его данные (рост, масса тела, возраст, уровень физической активности) и цели (похудение, набор массы, поддержание оптимального веса, сбалансированное питание)
2. План питания на день (завтрак, обед, ужин, перекусы) с указанием необходимых КБЖУ, учитывая цели пользователя
3. Подсчет КБЖУ в продуктах, которые пользователь указывает в своем плане питания в граммах
4. Варианты завтраков, обедов, ужинов, перекусов (фото)
5. Рецепты блюд с фото
6. Питьевой режим (пользователь указывает в своем плане питания кол-во выпитой в течение дня воды)
7. Режим физической активности (пользователь вводит данные о тренировках, времени)

Пользователь получает отчет в цифрах и диаграммах с рекомендациями по результату отчета, что можно улучшить завтра.
Пользователю предлагаются варианты завтраков, обедов, ужинов, перекусов с фото на завтра.


Представления и их назначение

1. При входе пользователь попадает на стартовую страницу с меню "Мои данные", "Дневник питания", "Отчеты и рекомендации".

    1.1. Вкладка "Мои данные" предлагает пользователю ввести свои данные (рост, масса тела, возраст, уровень физической активности) и цель (похудение, набор массы, поддержание оптимального веса).
        Данные обрабатываются. Вычисляется ИМТ (индекс массы тела) и сравнивается с данными ВОЗ.
        Если цели пользователя не противоречат данным ВОЗ, проводится расчет суточной нормы БЖУ, калорий и воды.
        Если цели пользователя противоречат данным ВОЗ, то пользователь получает предупреждение.

    1.2. Вкладка "Дневник питания" содержит меню "Завтрак", "Обед", "Ужин", "Перекус", "Питьевой режим", "Физическая нагрузка".
        При переходе в "Завтрак", "Обед", "Ужин", "Перекус" появляется строка поиска, где можно найти нужный продукт и указать кол-во съеденного в граммах. Выбранные продукты добавляются в список соответствующего приема пищи.
        При переходе в "Питьевой режим" пользователь указывает количество выпитой в течение дня воды в стаканах (1 стакан = 250 мл).
        При переходе в "Физическая нагрузка" появляется строка поиска, где пользователь выбирает тип нагрузки и длительность.

    1.3. Во вкладке "Отчеты и рекомендации" отчеты по калориям, БЖУ, водному балансу и нагрузкам сохраняются в календарь. По результату отчета даются краткие рекомендациями, что можно улучшить завтра. Пользователю предлагаются варианты завтраков, обедов, ужинов, перекусов с фото на завтра.


Структура меню

    Мои данные ->
        Рост
        Масса тела
        Возраст
        Уровень физической активности

    Дневник питания ->
        Завтрак
        Обед
        Ужин
        Перекус
        Питьевой режим
        Физическая нагрузка

    Отчеты и рекомендации ->
        Количество калорий в цифрах за день
        Диаграмма БЖУ
        Диаграмма водного баланса
        Количество сожжённых калорий
        Краткие рекомендациям
        Варианты завтраков, обедов, ужинов, перекусов с фото
